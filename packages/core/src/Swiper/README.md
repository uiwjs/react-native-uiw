## Props

属性 | 说明 | 类型 | 默认值
----|-----|------|------
| dataSource   | 数据源         | Array     | [ ] |
| width        | 宽度           |  Number   |屏幕宽度|
| height       | 高度           | Number    |  130 |
| time         | 执行时间       | Number    | 3000（3s） |
| borderRadius | 圆角边框       | Number    |  0 |
| autoplay     | 是否开启定时器  | Boolean   | true|
| dotStyle     | 圆点类型 ( dot : 圆点， block : 方块 )       | String    | dot |
