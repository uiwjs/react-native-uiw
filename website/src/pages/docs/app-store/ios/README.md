APP上架前需注册开发者账号，创建 **`证书请求文件（CSR文件）`** 、 **`发布证书 （CER）`** 及 **`Provisioning Profiles配置文件 （PP文件）`** ,创建完成后，即可准备将已完成的项目进行提交上架。
